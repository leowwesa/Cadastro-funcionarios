# 🚀 Sistema de gerenciamento de funcionarios

É um sistema para o controle de funcionários de uma empresa,
com as funcionalidades utilizadas no dia a dia da gestão de funcionarios de uma empresa

## ✨ Funcionalidades
Contendo todas as opções basicas de um CRUD sem a integração ao banco de dados ainda.
São opções simples que no momento, salvam os usuarios cadastrados em uma lista, a ideia central é integrar esse sistema com o mysql, 
criei um sistema que tem as seguintes opções
1. Criar Funcionário
2. Listar Funcionário
3. Atualizar Funcionário 
4. Deletar Funcionário
0. Sair

## 🛠️ Tecnologias Utilizadas

- Utilizei Java na versão 17+ criando o projeto com classes e herança, não precisando instalar dependecias, utilizei apenas o pacote padrão do java
- java.util.(nome da biblioteca exemplo abaixo)
- java.util.Scanner;
- java.util.List;

- ## ▶️ Como Rodar

1. Abra o projeto no IntelliJ ou qualquer IDE Java.  
2. Compile e execute a classe `Main.java`.  
3. Teste adicionando, listando, atualizando ou removendo funcionários.
